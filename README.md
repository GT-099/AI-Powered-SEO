# AI-Powered SEO Content Production — Research Project

This repository collects recent content (YouTube transcripts and LinkedIn posts)
from 10 active practitioners in AI-powered SEO content production, for research
and analysis purposes.

## Why these experts?
Each expert was selected because they:
- Actively practice AI-powered SEO/content work (not just write about it)
- Have publicly accessible recent content (last 3-6 months)
- Cover a mix of platforms (YouTube, LinkedIn, newsletters)

See `research/sources.md` for the full list with rationale for each.

## Structure
- `research/sources.md` — list of 10 experts with links and annotations
- `research/youtube-transcripts/` — transcripts organized by author
- `research/linkedin-posts/` — posts organized by author
- `research/other/` — newsletters, podcast transcripts, articles
